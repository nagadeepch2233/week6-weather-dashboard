 Weather Dashboard

 A terminal-based weather dashboard using OpenWeatherMap API.

 Features
 
- Current weather
- 5-day forecast
- Emoji indicators
- Caching (10 minutes)
- Search city
- Refresh data

 Setup

1. Clone repository
2. Create virtual environment
3. Install dependencies:
   pip install -r requirements.txt
4. Create .env file from .env.example
5. Run:
   python -m weather_app.main
